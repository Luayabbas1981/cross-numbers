# crossed-numbers

## Description
Crossed numbers is a puzzle game where players must place arithmetic operators in the correct positions to make equations valid.

## How to Play
1. Start the game by launching the application.
2. Players are presented with a grid of numbers and empty squares representing operators (+, -, *, /).
3. The objective is to fill in the empty squares with the correct operators to make equations valid both horizontally and vertically.
4. Once all operators are placed, the player can check their solution for correctness.
